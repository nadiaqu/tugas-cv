## 1. Deskripsi Aplikasi
Aplikasi ini adalah sebuah **Digital Curriculum Vitae (CV)** berbasis web yang dibangun menggunakan **ReactJS**. Aplikasi ini dirancang dengan konsep *Single Page Application* (SPA) yang responsif dan modern, mengadopsi gaya tampilan *professional network* (seperti LinkedIn).

**Fitur Utama:**
* **Profile Card:** Menampilkan foto profil, banner, nama, headline, dan lokasi.
* **About Section:** Ringkasan singkat profil profesional.
* **Experience List:** Daftar pengalaman kerja/organisasi dengan layout kronologis.
* **Skills Badge:** Daftar keahlian teknis yang dimiliki.
* **Data Driven:** Semua konten diambil dari file JSON terpisah (`dataMahasiswa.json`) sehingga mudah diperbarui tanpa mengubah kode program.

**Teknologi yang Digunakan:**
* ReactJS + Vite
* CSS3 (Custom Styling)
* Git & GitHub

## 3. Cara Menjalankan Project (Local)
Jika ingin menjalankan project ini di komputer lokal, ikuti langkah berikut:

1.  **Clone Repository**
    ```bash
    git clone [LINK_REPO_GITHUB_KAMU]
    ```

2.  **Masuk ke Folder Project**
    ```bash
    cd cv-mahasiswa
    ```

3.  **Install Dependencies**
    ```bash
    npm install
    ```

4.  **Jalankan Server Development**
    ```bash
    npm run dev
    ```
5.  Buka browser dan akses `http://localhost:5173`
